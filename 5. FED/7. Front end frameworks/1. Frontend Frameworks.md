## Component lifecycle

### React

Функциональные компоненты — через Hooks. [[1. React/2. React Hooks/2. React Hooks]]

```javascript
function Component() {
  const [state, setState] = useState(0);

  useEffect(() => {
    // mount + update
    return () => { /* unmount */ };
  }, [dep]);

  useEffect(() => {
    // только mount
  }, []);
}
```

| Фаза | Hook |
| :-- | :-- |
| Mount | `useEffect(() => {}, [])` |
| Update | `useEffect(() => {}, [deps])` |
| Unmount | `return () => {}` в useEffect |
| До paint | `useLayoutEffect` |

### Angular

Жизненный цикл через интерфейсы и декораторы.

```typescript
@Component({...})
export class MyComponent implements OnInit, OnDestroy {
  ngOnInit() { /* после первого проверки */ }
  ngAfterViewInit() { /* после инициализации view */ }
  ngOnDestroy() { /* перед удалением */ }
}
```

| Hook | Когда |
| :-- | :-- |
| `ngOnChanges` | Изменение @Input |
| `ngOnInit` | Инициализация |
| `ngDoCheck` | Каждая проверка изменений |
| `ngAfterViewInit` | View готов |
| `ngOnDestroy` | Разбор компонента |

### Vue

```javascript
// Composition API
import { onMounted, onUpdated, onUnmounted } from 'vue';

export default {
  setup() {
    onMounted(() => { /* mount */ });
    onUpdated(() => { /* update */ });
    onUnmounted(() => { /* unmount */ });
  }
}
```

```vue
<!-- Options API -->
<script>
export default {
  mounted() { },
  updated() { },
  unmounted() { }
}
</script>
```

## State management

### React

- **Локальный** — `useState`, `useReducer` [[1. React/2. React Hooks/2.2 useState.md]]
- **Глобальный** — Context API [[1. React/1. React Core Concepts/1.9 Context API.md]]
- **Сторонние** — Redux Toolkit, Zustand, Jotai, TanStack Query (server state)

```javascript
const [state, setState] = useState(0);
const [state, dispatch] = useReducer(reducer, initial);
const value = useContext(MyContext);
```

### Angular

- **Локальный** — свойства компонента
- **Глобальный** — Services (singleton), RxJS
- **Сторонние** — NgRx, Akita

```typescript
@Injectable({ providedIn: 'root' })
export class StateService {
  private state = new BehaviorSubject(0);
  readonly state$ = this.state.asObservable();
}
```

### Vue

- **Локальный** — `ref`, `reactive`
- **Глобальный** — Pinia (официальный), Vuex (legacy)
- **Composition** — `provide/inject`

```javascript
const count = ref(0);
const state = reactive({ items: [] });
```

## Performance optimizations

### React

[[1. React/12. Performance/6. Performance]] · [[5. FED/3. JavaScript/5. Performance/2. virtual DOM]]

| Приём | Назначение |
| :-- | :-- |
| `React.memo` | Пропуск ре-рендера при неизменных props |
| `useMemo` | Мемоизация вычислений |
| `useCallback` | Мемоизация колбэков |
| `useTransition` | Неблокирующие обновления |
| `useDeferredValue` | Отложенное обновление |
| Code splitting | `lazy()`, динамический import |
| Virtualization | react-window для длинных списков |

```javascript
const MemoChild = memo(Child);
const value = useMemo(() => compute(a, b), [a, b]);
const fn = useCallback(() => doSomething(), [dep]);
```

### Angular

| Приём | Назначение |
| :-- | :-- |
| `OnPush` | Change detection только при изменении @Input |
| `trackBy` | Стабильные ключи в *ngFor |
| `async` pipe | Автоматическая отписка |
| Lazy loading | `loadChildren` в роутере |
| Pure pipes | Мемоизация в pipe |

```typescript
@Component({ changeDetection: ChangeDetectionStrategy.OnPush })
```

```html
<div *ngFor="let item of items; trackBy: trackById">{{ item.name }}</div>
```

### Vue

| Приём | Назначение |
| :-- | :-- |
| `v-once` | Один рендер |
| `v-memo` | Условное обновление |
| `shallowRef` | Реактивность без глубокого сравнения |
| `defineAsyncComponent` | Lazy loading |
| Virtualization | vue-virtual-scroller |

```vue
<div v-memo="[item.id === selected]">{{ item.name }}</div>
```

## Сравнение

| Аспект | React | Angular | Vue |
| :-- | :-- | :-- | :-- |
| Кривая обучения | Средняя | Высокая | Низкая |
| Размер | ~40kb | ~150kb+ | ~35kb |
| Шаблоны | JSX | HTML + директивы | SFC, HTML |
| Реактивность | Явный setState | Zone.js | Proxy |
| Роутинг | React Router | Встроен | Vue Router |
| SSR | Next.js | Angular Universal | Nuxt |
