## Flexbox

**Зачем:** одномерная раскладка — выравнивание элементов в строке или столбце без float и таблиц. Подходит для навигации, кнопок, карточек в ряд, центрирования.

| Свойство | Назначение |
| :-- | :-- |
| `flex-direction` | `row`, `column`, `row-reverse`, `column-reverse` |
| `justify-content` | Выравнивание по главной оси |
| `align-items` | Выравнивание по поперечной оси |
| `flex-wrap` | Перенос строк |
| `gap` | Расстояние между элементами |

**На дочерних:** `flex-grow`, `flex-shrink`, `flex-basis`, `flex` (shorthand).

```css
/* Навигация: пункты в ряд, по краям */
.nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 1rem;
}
```

## Grid

**Зачем:** двумерная раскладка — колонки и строки одновременно. Подходит для макета страницы, сетки карточек, сложных областей (header, sidebar, content).

| Свойство | Назначение |
| :-- | :-- |
| `grid-template-columns/rows` | Размеры колонок/строк |
| `grid-template-areas` | Именованные области |
| `gap` | Расстояние между ячейками |
| `place-items` | Выравнивание содержимого ячейки |

**На дочерних:** `grid-column`, `grid-row`, `grid-area`.

```css
/* Макет: header на всю ширину, sidebar + content */
.page {
  display: grid;
  grid-template-areas:
    "header header"
    "sidebar main";
  grid-template-columns: 200px 1fr;
  gap: 1rem;
}
.page-header { grid-area: header; }
.page-sidebar { grid-area: sidebar; }
.page-main { grid-area: main; }
```

## Flexbox vs Grid

- **Flexbox** — компоненты, навигация, выравнивание в строке/столбце
- **Grid** — макет страницы, карточки, таблицы
