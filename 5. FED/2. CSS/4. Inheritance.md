# 4. Inheritance

[[5. FED/2. CSS/3. Specificity]]

## Зачем нужна

Не дублировать стили для каждого потомка. Задать `color` или `font-size` на родителе — и все вложенные элементы получат их. Экономия кода и единообразие типографики.

## Определение

**Inheritance** — наследование значений свойств от родителя к потомкам. Не все свойства наследуются.

## Наследуемые

`color`, `font-*`, `line-height`, `text-align`, `visibility`, `cursor` и др.

## Ненаследуемые

`background`, `border`, `margin`, `padding`, `width`, `height`, `display` и др.

## Пример

```css
/* Все тексты внутри .article наследуют цвет и размер шрифта */
.article {
  color: #333;
  font-size: 1rem;
  line-height: 1.5;
}
.article h2 { font-size: 1.5em; } /* em относительно родителя */
```

## Управление

```css
.child { color: inherit; }   /* явно наследовать */
.child { all: unset; }       /* сбросить всё */
.child { all: revert; }      /* вернуть к user-agent */
```
