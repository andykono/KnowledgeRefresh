## Зачем нужен

Один макет для разных экранов — мобильный, планшет, десктоп. Меньше горизонтального скролла, читаемый текст, удобные кнопки. Требование для современного веба и SEO.

## Media queries

```css
@media (min-width: 768px) { /* планшет и выше */ }
@media (max-width: 767px) { /* мобильный */ }
@media (prefers-reduced-motion: reduce) { /* доступность */ }
```

## Относительные единицы

| Единица | Относится к |
| :-- | :-- |
| `rem` | `font-size` корня |
| `em` | `font-size` родителя |
| `vw`, `vh` | 1% viewport |
| `%` | родитель |

## Пример: mobile-first

```css
/* Базовые стили — мобильный */
.card { padding: 1rem; font-size: 0.9rem; }

/* Планшет и выше */
@media (min-width: 768px) {
  .card { padding: 1.5rem; font-size: 1rem; }
}

/* Десктоп */
@media (min-width: 1200px) {
  .card { padding: 2rem; }
}
```

## Контейнеры

```css
.container {
  width: min(100% - 2rem, 1200px);
  margin-inline: auto;
}
```
