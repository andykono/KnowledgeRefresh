## Зачем нужны

Короткий синтаксис, лексический `this` (без `bind`), удобны для колбэков, `.map()`, `.filter()`, обработчиков.

## Синтаксис

```javascript
const add = (a, b) => a + b;
const double = x => x * 2;
const fn = () => ({ key: 1 });  // объект — в скобках

// Один параметр — скобки опциональны
arr.map(x => x * 2);
```

## this

Стрелочная функция **не имеет своего `this`** — берёт из внешней области. [[5. FED/3. JavaScript/1. Core concepts/1. this]]

```javascript
const obj = {
  name: 'Test',
  fn: function () {
    setTimeout(() => console.log(this.name), 0);  // "Test" — this из fn
  }
};
```

## Ограничения

- Нет `arguments` (использовать rest: `(...args) => ...`)
- Нельзя использовать как конструктор (`new`)
