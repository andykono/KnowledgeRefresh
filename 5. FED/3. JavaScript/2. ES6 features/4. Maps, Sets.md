# 4. Maps, Sets

[[5. FED/3. JavaScript/2. ES6 features/3. destructuring, spread-rest]]

## Зачем нужны

**Map** — ключ-значение, ключи любого типа (в отличие от объекта). **Set** — уникальные значения. Быстрый поиск, подсчёт, дедупликация.

## Map

```javascript
const map = new Map();
map.set('key', 'value');
map.set(1, 'one');
map.set({ id: 1 }, 'obj as key');

map.get('key');      // "value"
map.has('key');      // true
map.delete('key');
map.size;

// Итерация
for (const [k, v] of map) { }
map.forEach((v, k) => { });
```

## Set

```javascript
const set = new Set([1, 2, 2, 3]);  // Set(3) { 1, 2, 3 }

set.add(4);
set.has(2);   // true
set.delete(2);
set.size;

// Дедупликация массива
const unique = [...new Set(arr)];
```

## Map vs Object

| | Object | Map |
| :-- | :-- | :-- |
| Ключи | string/symbol | любой тип |
| Порядок | не гарантирован (ES2015+) | порядок вставки |
| size | — | `map.size` |
| Итерация | `Object.keys()` | `for...of` напрямую |
