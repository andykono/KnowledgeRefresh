# 3. destructuring, spread/rest operators

[[5. FED/3. JavaScript/2. ES6 features/4. Maps, Sets]]

## Зачем нужны

**Destructuring** — извлечение значений из массивов/объектов в переменные. **Spread** (`...`) — разворачивание. **Rest** (`...`) — сбор оставшихся аргументов/элементов. Меньше кода, удобная работа с API.

## Destructuring

```javascript
// Массив
const [a, b, ...rest] = [1, 2, 3, 4];  // a=1, b=2, rest=[3,4]

// Объект
const { name, age = 18 } = user;
const { name: userName } = user;  // переименование

// В параметрах
function fn({ x, y }) {
  return x + y;
}
```

## Spread

```javascript
const arr = [1, 2, 3];
const copy = [...arr];

const obj = { a: 1, b: 2 };
const merged = { ...obj, b: 3 };  // { a: 1, b: 3 }

// Вызов функции
Math.max(...[1, 5, 3]);
```

## Rest

```javascript
function sum(...nums) {
  return nums.reduce((a, b) => a + b, 0);
}
sum(1, 2, 3);  // 6

const [first, ...others] = [1, 2, 3];  // first=1, others=[2,3]
```
