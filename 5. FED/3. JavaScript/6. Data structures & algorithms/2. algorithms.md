## Sorting

### Для чего

Упорядочивание данных. Поиск, отображение, бинарный поиск требует отсортированный массив.

### Quick Sort

**Для чего:** O(n log n) в среднем, in-place. Универсальный выбор.

```javascript
function quickSort(arr) {
  if (arr.length <= 1) return arr;
  const pivot = arr[Math.floor(arr.length / 2)];
  const left = arr.filter(x => x < pivot);
  const mid = arr.filter(x => x === pivot);
  const right = arr.filter(x => x > pivot);
  return [...quickSort(left), ...mid, ...quickSort(right)];
}
```

### Merge Sort

**Для чего:** O(n log n) в любом случае, стабильный. Когда нужна предсказуемость.

```javascript
function mergeSort(arr) {
  if (arr.length <= 1) return arr;
  const mid = Math.floor(arr.length / 2);
  const left = mergeSort(arr.slice(0, mid));
  const right = mergeSort(arr.slice(mid));
  return merge(left, right);
}

function merge(a, b) {
  const result = [];
  let i = 0, j = 0;
  while (i < a.length && j < b.length) {
    result.push(a[i] <= b[j] ? a[i++] : b[j++]);
  }
  return [...result, ...a.slice(i), ...b.slice(j)];
}
```

### Сравнение

| Алгоритм | Время (среднее) | Время (худший) | Память | Стабильный |
| :-- | :-- | :-- | :-- | :-- |
| Quick Sort | O(n log n) | O(n²) | O(log n) | нет |
| Merge Sort | O(n log n) | O(n log n) | O(n) | да |
| Bubble Sort | O(n²) | O(n²) | O(1) | да |

## Searching

### Linear Search

**Для чего:** неотсортированный массив. O(n).

```javascript
function linearSearch(arr, target) {
  for (let i = 0; i < arr.length; i++) {
    if (arr[i] === target) return i;
  }
  return -1;
}
```

### Binary Search

**Для чего:** отсортированный массив. O(log n).

```javascript
function binarySearch(arr, target) {
  let left = 0, right = arr.length - 1;
  while (left <= right) {
    const mid = Math.floor((left + right) / 2);
    if (arr[mid] === target) return mid;
    if (arr[mid] < target) left = mid + 1;
    else right = mid - 1;
  }
  return -1;
}
```

### Для чего какой

- **Linear** — неотсортированные данные, поиск по условию
- **Binary** — отсортированный массив, поиск границы (lower_bound)
