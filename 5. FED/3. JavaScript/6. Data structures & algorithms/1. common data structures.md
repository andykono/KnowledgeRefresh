## Arrays

**Для чего:** упорядоченная коллекция, доступ по индексу O(1). Списки, очереди, хранение данных.

```javascript
const arr = [1, 2, 3];
arr[0];           // доступ O(1)
arr.push(4);      // в конец O(1)
arr.pop();        // с конца O(1)
arr.unshift(0);   // в начало O(n) — сдвиг
arr.splice(1, 0, 'x');  // вставка O(n)
```

## Strings

**Для чего:** последовательность символов. Поиск подстроки, парсинг, валидация.

```javascript
const s = 'hello';
s.includes('ell');     // true
s.indexOf('l');       // 2
s.slice(1, 4);        // 'ell'
s.split('').reverse().join('');  // разворот
```

## Linked Lists

**Для чего:** цепочка узлов (value + next). Вставка/удаление в начало O(1). Когда не нужен случайный доступ, часто меняется структура.

```javascript
class Node {
  constructor(value, next = null) {
    this.value = value;
    this.next = next;
  }
}

const head = new Node(1, new Node(2, new Node(3)));
// 1 -> 2 -> 3
```

Вставка в начало: `new Node(x, head)`. Удаление: перекинуть `next`.

## Trees

**Для чего:** иерархия. DOM, JSON, файловая система, BST для поиска.

### Binary Search Tree (BST)

Левый потомок < родитель < правый. Поиск/вставка O(log n) в среднем.

```javascript
class TreeNode {
  constructor(val) {
    this.val = val;
    this.left = null;
    this.right = null;
  }
}

function search(root, target) {
  if (!root || root.val === target) return root;
  return target < root.val ? search(root.left, target) : search(root.right, target);
}
```

### Обход

```javascript
// In-order: левый -> корень -> правый (сортировка для BST)
function inorder(node) {
  if (!node) return [];
  return [...inorder(node.left), node.val, ...inorder(node.right)];
}
```

## Graphs

**Для чего:** узлы и рёбра. Соцсети, карты, зависимости.

### Представление

**Adjacency list** — массив списков соседей. Экономно по памяти.

```javascript
const graph = {
  A: ['B', 'C'],
  B: ['A', 'D'],
  C: ['A', 'D'],
  D: ['B', 'C']
};
```

**Adjacency matrix** — матрица N×N. Проверка ребра O(1), но O(n²) памяти.

### BFS (поиск в ширину)

```javascript
function bfs(graph, start) {
  const visited = new Set();
  const queue = [start];
  visited.add(start);

  while (queue.length) {
    const node = queue.shift();
    for (const neighbor of graph[node] || []) {
      if (!visited.has(neighbor)) {
        visited.add(neighbor);
        queue.push(neighbor);
      }
    }
  }
  return visited;
}
```

### DFS (поиск в глубину)

```javascript
function dfs(graph, node, visited = new Set()) {
  if (visited.has(node)) return;
  visited.add(node);
  for (const neighbor of graph[node] || []) {
    dfs(graph, neighbor, visited);
  }
  return visited;
}
```
