# 3. event delegation

[[5. FED/3. JavaScript/3. DOM manipulation/2. event handling]] · [[5. FED/3. JavaScript/3. DOM manipulation/1. Efficiently interacting with the DOM]]

## Зачем нужна

**Event delegation** — один слушатель на родителе вместо множества на дочерних. Событие всплывает (bubbling), родитель получает его и по `e.target` определяет, какой дочерний элемент сработал.

## Преимущества

- Меньше слушателей — меньше памяти
- Динамический контент — новые элементы автоматически «подхватываются»
- Меньше привязок к DOM при инициализации

## Как работает

Событие **всплывает** от target к document. Родитель ловит событие и проверяет `e.target`.

```javascript
const list = document.getElementById('list');

list.addEventListener('click', (e) => {
  const item = e.target.closest('li');  // ближайший li вверх по дереву
  if (!item) return;

  console.log('Clicked:', item.dataset.id);
});
```

## Важно: e.target vs e.currentTarget

- `e.target` — элемент, на который реально кликнули (может быть вложенный `span`, `strong`)
- `e.currentTarget` — элемент со слушателем (родитель)

```html
<ul id="list">
  <li><strong>Item 1</strong></li>
</ul>
```

Клик по «Item 1»: `e.target` = `<strong>`, `e.currentTarget` = `<ul>`. Поэтому используют `e.target.closest('li')` для поиска нужного предка.

## Пример: список с удалением

```javascript
list.addEventListener('click', (e) => {
  if (e.target.matches('button.delete')) {
    const item = e.target.closest('li');
    item?.remove();
  }
});
```

`matches(selector)` — проверяет, подходит ли элемент под селектор.

## Пример: таблица

```javascript
table.addEventListener('click', (e) => {
  const cell = e.target.closest('td');
  if (!cell) return;

  const row = cell.closest('tr');
  const rowIndex = row.rowIndex;
  const colIndex = cell.cellIndex;
  // обработка
});
```

## Связь с производительностью

Один слушатель вместо N — меньше работы при [[5. FED/3. JavaScript/3. DOM manipulation/1. Efficiently interacting with the DOM|инициализации DOM]], меньше нагрузка при скролле/частых событиях.
