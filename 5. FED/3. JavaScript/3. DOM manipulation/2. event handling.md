# 2. event handling

[[5. FED/3. JavaScript/1. Core concepts/4. event loop]] · [[5. FED/3. JavaScript/3. DOM manipulation/1. Efficiently interacting with the DOM]] · [[5. FED/3. JavaScript/3. DOM manipulation/3. event delegation]]

## Зачем понимать

События — основа интерактивности. Нужно знать фазы (capturing/bubbling), объект события, отмену поведения, разницу между способами подписки.

## addEventListener

```javascript
element.addEventListener('click', handler, options);
```

**options:**
- `capture: true` — фаза capturing (по умолчанию `false` — bubbling)
- `once: true` — снять после первого срабатывания
- `passive: true` — не будет `preventDefault()`, браузер может оптимизировать scroll

## Фазы события

1. **Capturing** — от `window` вниз к target
2. **Target** — на самом элементе
3. **Bubbling** — от target вверх к `window`

```html
<div id="outer">
  <button id="inner">Click</button>
</div>
```

При клике на button: capturing (div → button) → target (button) → bubbling (button → div).

## Объект события (e)

| Свойство/метод | Назначение |
| :-- | :-- |
| `e.target` | Элемент, на котором произошло событие |
| `e.currentTarget` | Элемент, на котором висит слушатель (с `this` совпадает) |
| `e.type` | Тип события (`'click'`, `'keydown'`) |
| `e.preventDefault()` | Отменить действие по умолчанию (submit, link) |
| `e.stopPropagation()` | Остановить всплытие (bubbling) |
| `e.stopImmediatePropagation()` | Остановить всплытие и другие слушатели на этом элементе |

```javascript
link.addEventListener('click', (e) => {
  e.preventDefault();  // не переходить по href
});

form.addEventListener('submit', (e) => {
  e.preventDefault();  // не отправлять форму
});
```

## addEventListener vs onclick

| | addEventListener | onclick (и др.) |
| :-- | :-- | :-- |
| Несколько слушателей | да | нет (перезапись) |
| Опции (capture, once) | да | нет |
| Удаление | `removeEventListener(fn)` | присвоить `null` |

```javascript
const handler = () => console.log('click');
el.addEventListener('click', handler);
el.removeEventListener('click', handler);  // нужна та же ссылка на функцию
```
