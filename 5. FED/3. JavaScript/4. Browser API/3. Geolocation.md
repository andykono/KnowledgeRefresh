# 3. Geolocation

[[5. FED/3. JavaScript/4. Browser API/4. other related]]

## Зачем нужен

Получение координат пользователя (GPS, Wi‑Fi, IP). Карты, ближайшие точки, персонализация.

## API

```javascript
navigator.geolocation.getCurrentPosition(
  (position) => {
    const { latitude, longitude } = position.coords;
    const accuracy = position.coords.accuracy;  // метры
  },
  (error) => {
    // 1: PERMISSION_DENIED, 2: POSITION_UNAVAILABLE, 3: TIMEOUT
    console.error(error.code, error.message);
  },
  { enableHighAccuracy: true, timeout: 10000, maximumAge: 0 }
);
```

## Отслеживание (watchPosition)

```javascript
const watchId = navigator.geolocation.watchPosition(
  (position) => console.log(position.coords),
  (error) => console.error(error)
);

navigator.geolocation.clearWatch(watchId);  // остановить
```

## Опции

| Опция | Назначение |
| :-- | :-- |
| `enableHighAccuracy` | GPS вместо IP/Wi‑Fi (медленнее, точнее) |
| `timeout` | мс до ошибки TIMEOUT |
| `maximumAge` | использовать кэш до N мс (0 = всегда свежие) |

## Важно

- Требует разрешения пользователя (HTTPS или localhost)
- `navigator.geolocation` может быть `undefined` — проверять наличие
