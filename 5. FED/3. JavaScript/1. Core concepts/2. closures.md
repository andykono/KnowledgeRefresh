## Зачем нужны

**Closure** (замыкание) — функция сохраняет доступ к переменным из внешней области видимости после её завершения. Нужны для: приватных данных, фабрик, колбэков, debounce/throttle.

## Определение

Функция + лексическое окружение (scope), в котором она была создана.

## Пример

```javascript
function createCounter() {
  let count = 0;  // «приватная» переменная
  return function () {
    return ++count;
  };
}

const counter = createCounter();
counter();  // 1
counter();  // 2
// count недоступен снаружи
```

## Stale closure

При отложенном вызове (setTimeout, Promise) замыкание может «захватить» устаревшее значение. [[5. FED/3. JavaScript/1. Core concepts/5. hoisting]]

```javascript
for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 0);  // 3 3 3  (var — одна на цикл)
}
for (let i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 0);  // 0 1 2  (let — новая на итерацию)
}
```

## Связь с this

Стрелочные функции не имеют `this` — берут из замыкания. [[5. FED/3. JavaScript/1. Core concepts/1. this]]
