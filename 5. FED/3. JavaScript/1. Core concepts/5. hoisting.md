# 5. hoisting

## Зачем понимать

**Hoisting** — «поднятие» объявлений в начало области видимости до выполнения кода. Объясняет поведение `var` до присваивания, разницу между `var`, `let`, `const`.

## var

Объявление поднимается, присваивание — нет. Область видимости — функция.

```javascript
console.log(x);  // undefined (не ReferenceError)
var x = 5;
console.log(x);  // 5

// Эквивалентно:
var x;
console.log(x);
x = 5;
console.log(x);
```

## let и const

Поднимаются, но **временная мёртвая зона (TDZ)** — до строки объявления обращение вызовет `ReferenceError`.

```javascript
console.log(a);  // ReferenceError
let a = 1;

console.log(b);  // ReferenceError
const b = 2;
```

## Scope

| Ключевое слово | Scope | Повторное объявление |
| :-- | :-- | :-- |
| `var` | function | да |
| `let` | block `{}` | нет |
| `const` | block `{}` | нет |

```javascript
for (var i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 0);  // 3 3 3
}
for (let i = 0; i < 3; i++) {
  setTimeout(() => console.log(i), 0);  // 0 1 2  (новая i на итерацию)
}
```
