## Core Principles

### 1. Components are Functions
```typescript
// Functional component
function Greeting({ name }) {
  return <h1>Hello, {name}!</h1>;
}
```

### 2. One-Way Data Flow
- **Props flow DOWN:** Parent → Child
- **Events flow UP:** Child → Parent
```typescript
// Parent
<Child value={count} onChange={setCount} />

// Child receives props, calls callback
<button onClick={() => onChange(count + 1)}>+</button>
```

### 3. State Causes Re-renders
When state changes, React re-renders the component and its children.

### 4. Keys Identify List Items
```typescript
{items.map(item => (
  <div key={item.id}>  {/* Unique key! */}
    {item.name}
  </div>
))}
```

### 5. Refs Access the DOM
Use only when you really need to (form focus, text selection, media playback).
