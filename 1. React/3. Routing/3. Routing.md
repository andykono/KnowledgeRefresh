## Core Concepts

### Client-Side Routing
React Router keeps the UI in sync with the URL without full page reloads. Use `BrowserRouter` for HTML5 History API (clean URLs).

### Navigation Patterns
- **Declarative:** `<Link>`, `<NavLink>` for user clicks
- **Programmatic:** `useNavigate()` for code-triggered navigation
- **Redirect:** `<Navigate>` for auth guards and redirects

### Key Hooks
| Hook | Use For |
| :-- | :-- |
| useParams | Dynamic route segments (`:id`) |
| useSearchParams | Query string (`?q=...`) |
| useLocation | Path, search, state |
| useNavigate | Programmatic navigation |
