# Throttle

[[4. Common/2. Debounce/1. Debounce]] · [[4. Common/2. Debounce/2. ReactHooks]] · [[4. Common/2. Debounce/3. Examples]]

## Определение

**Throttle** (ограничение частоты) — техника, которая гарантирует, что функция выполняется **не чаще чем раз в N миллисекунд**. В отличие от debounce, первый вызов срабатывает сразу.

### Зачем нужен throttle?

- **Scroll** — обновлять позицию, подгружать контент (infinite scroll)
- **Resize** — реагировать на изменение окна с ограничением
- **Drag** — обрабатывать перемещение без лагов
- **Mousemove** — трекинг без перегрузки

### Debounce vs Throttle

- **Debounce** — выполняет функцию **после** паузы в вызовах. Подходит для поиска, автосохранения, валидации.
- **Throttle** — выполняет функцию **не чаще** чем раз в N мс. Подходит для scroll, resize, drag.

## 1. Простой пример на JavaScript

```javascript
function throttle(fn, limit) {
  let inThrottle;

  return function (...args) {
    if (!inThrottle) {
      fn.apply(this, args);
      inThrottle = true;
      setTimeout(() => (inThrottle = false), limit);
    }
  };
}

// Использование
const handleScroll = throttle(() => {
  console.log('Позиция скролла:', window.scrollY);
}, 100);

window.addEventListener('scroll', handleScroll);
```

### Объяснение

- **`inThrottle`** — флаг, указывающий, находимся ли мы в периоде ограничения.
- **`if (!inThrottle)`** — функция выполняется только если мы не в периоде ограничения.
- **Первый вызов сразу** — в отличие от debounce.
- **`setTimeout(() => (inThrottle = false), limit)`** — через `limit` мс снимаем ограничение.

При быстром скролле вызов происходит раз в 100 мс, а не при каждом событии.

## 2. Более надежная версия (с timeoutId)

```javascript
function throttle(fn, limit) {
  let inThrottle;
  let lastFunc;
  let lastRan;

  return function (...args) {
    const context = this;

    if (!lastRan) {
      fn.apply(context, args);
      lastRan = Date.now();
    } else {
      clearTimeout(lastFunc);
      lastFunc = setTimeout(() => {
        if ((Date.now() - lastRan) >= limit) {
          fn.apply(context, args);
          lastRan = Date.now();
        }
      }, limit - (Date.now() - lastRan));
    }
  };
}
```

### Преимущества

- Гарантирует последний вызов, даже если событие произойдет в конце окна throttle
- Более точный контроль времени

## 3. TypeScript версия

```typescript
function throttle<T extends (...args: unknown[]) => unknown>(
  fn: T,
  limit: number
): (...args: Parameters<T>) => void {
  let inThrottle = false;

  return function (this: ThisParameterType<T>, ...args: Parameters<T>) {
    if (!inThrottle) {
      fn.apply(this, args);
      inThrottle = true;

      setTimeout(() => {
        inThrottle = false;
      }, limit);
    }
  };
}

// Использование
const handleScroll = throttle((y: number) => {
  console.log('Позиция скролла:', y);
}, 100);

window.addEventListener('scroll', () => handleScroll(window.scrollY));
```

## 4. React Hook useThrottle

```tsx
import { useCallback, useRef } from 'react';

function useThrottle<T extends (...args: unknown[]) => unknown>(
  callback: T,
  limit: number
): (...args: Parameters<T>) => void {
  const inThrottleRef = useRef(false);
  const callbackRef = useRef(callback);

  callbackRef.current = callback;

  return useCallback(
    (...args: Parameters<T>) => {
      if (!inThrottleRef.current) {
        callbackRef.current(...args);
        inThrottleRef.current = true;

        setTimeout(() => {
          inThrottleRef.current = false;
        }, limit);
      }
    },
    [limit]
  );
}

// Использование в компоненте
function ScrollToTop() {
  const handleScroll = useThrottle(() => {
    const scrollTop = window.scrollY;
    console.log('Позиция скролла:', scrollTop);

    // Показать кнопку "вернуться вверх" если скроллили вниз
    const button = document.getElementById('scroll-to-top');
    if (button) {
      button.style.display = scrollTop > 300 ? 'block' : 'none';
    }
  }, 100);

  useEffect(() => {
    window.addEventListener('scroll', handleScroll);
    return () => window.removeEventListener('scroll', handleScroll);
  }, [handleScroll]);

  return (
    <button
      id="scroll-to-top"
      onClick={() => window.scrollTo({ top: 0, behavior: 'smooth' })}
    >
      ⬆️ Наверх
    </button>
  );
}
```

## Практические примеры

### Бесконечный скролл

```tsx
function InfiniteList() {
  const [items, setItems] = useState<Item[]>([]);
  const [page, setPage] = useState(1);
  const [loading, setLoading] = useState(false);

  const loadMore = useCallback(async () => {
    setLoading(true);
    const res = await fetch(`/api/items?page=${page}`);
    const newItems = await res.json();
    setItems((prev) => [...prev, ...newItems]);
    setPage((prev) => prev + 1);
    setLoading(false);
  }, [page]);

  const throttledLoadMore = useThrottle(loadMore, 500);

  const handleScroll = () => {
    const { scrollTop, scrollHeight, clientHeight } = document.documentElement;
    if (scrollHeight - scrollTop <= clientHeight + 100) {
      throttledLoadMore();
    }
  };

  useEffect(() => {
    window.addEventListener('scroll', handleScroll);
    return () => window.removeEventListener('scroll', handleScroll);
  }, [throttledLoadMore]);

  return (
    <>
      {items.map((item) => (
        <div key={item.id}>{item.name}</div>
      ))}
      {loading && <p>Загрузка...</p>}
    </>
  );
}
```

### Отслеживание позиции мыши

```tsx
function MouseTracker() {
  const [position, setPosition] = useState({ x: 0, y: 0 });

  const handleMouseMove = useThrottle((e: MouseEvent) => {
    setPosition({ x: e.clientX, y: e.clientY });
  }, 50);

  useEffect(() => {
    document.addEventListener('mousemove', handleMouseMove as any);
    return () =>
      document.removeEventListener('mousemove', handleMouseMove as any);
  }, [handleMouseMove]);

  return (
    <div>
      Позиция мыши: X={position.x}, Y={position.y}
    </div>
  );
}
```

### Обработка resize окна

```tsx
function ResponsiveComponent() {
  const [windowSize, setWindowSize] = useState({
    width: window.innerWidth,
    height: window.innerHeight,
  });

  const handleResize = useThrottle(() => {
    setWindowSize({
      width: window.innerWidth,
      height: window.innerHeight,
    });
  }, 200);

  useEffect(() => {
    window.addEventListener('resize', handleResize as any);
    return () => window.removeEventListener('resize', handleResize as any);
  }, [handleResize]);

  return (
    <div>
      Размер окна: {windowSize.width}x{windowSize.height}
    </div>
  );
}
```
